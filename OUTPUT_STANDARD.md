# 输出标准文档

## 1. 时间格式
- **统一使用美东时间**（ET/EDT）
- 格式：`HH:MM`

## 2. 交易明细表字段

| 字段 | 说明 |
|------|------|
| 日期 | MM-DD |
| 类型 | 正T/反T |
| 开仓时间 | 美东时间 HH:MM |
| 开仓动作 | 买入/卖出 |
| 开仓价 | $XXX.XX |
| 平仓时间 | 美东时间 HH:MM |
| 平仓动作 | 买入/卖出 |
| 平仓价 | $XXX.XX |
| 股数 | 固定100股 |
| 平仓原因 | 信号平仓/止损/强制平仓 |
| 毛盈亏 | $XXX.XX |
| 手续费 | $XX.XX（每笔$10 = 买卖各$5）|
| 净盈亏 | $XXX.XX |

## 3. 策略配置总结必须包含

- **初始现金**：$XX,XXX.XX
- **每次交易股数**：固定100股
- **每笔手续费**：$X.XX（明确说明买卖各多少）
- 其他关键参数：连续K线数、止损金额、时间限制等

## 4. 汇总格式

```
合计：毛盈亏 $XXX.XX，手续费 $XXX.XX，净盈亏 $XXX.XX
```
